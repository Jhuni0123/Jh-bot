# Flareon irc bot
`regex/부우*스터어*/` 라고 말하면 반응하는 봇 

## Features

- exchange rate ( !환율 [숫자] <통화명> [-> <통화명>] )

- word scoring ( !점수 <영단어> )

- fibonachicken ( !치킨 <숫자>[명] )

- fibonacci number ( !fib <숫자> )

- Codeforces contest list ( !코포 )

- Codeforces profile ( !코포 \<handle> )

- BOJ link ( !백준 <문제번호> )

## Run
```shell
python3 bot.py
```
